Comprehensive List of Exercises for Learning PySide6 Terminal (Non-GUI) Apps

Beginner Exercises

  1. Hello World Console App
    Create a basic terminal application that prints "Hello, World!" using QCoreApplication to initialize the Qt event loop, demonstrating the foundation for non-GUI PySide6 apps.
  2. Command-Line Argument Parsing
    Build a CLI tool that accepts and processes command-line flags (e.g., --verbose, --output-file) using QCommandLineParser, mimicking simple script utilities like file processors.
  3. Basic Event Loop Management
    Develop a console app that runs an event loop with QCoreApplication.exec() and handles a quit signal (e.g., via Ctrl+C), reflecting core event handling in long-running terminal tools.

Intermediate Exercises

  1. Signals and Slots in Console
    Implement a producer-consumer pattern in the terminal where one object emits a signal (e.g., data ready) and another slot processes it, useful for decoupled logic in CLI data pipelines.
  2. Timers for Periodic Tasks
    Create a countdown timer app that prints status updates at intervals using QTimer, simulating scheduled tasks in monitoring tools like system health checkers.
  3. File Reading and Writing
    Write a file copier utility that reads from one file and writes to another using QFile and QTextStream, handling errors for robust data migration scripts.
  4. JSON Data Handling
    Build a JSON validator and pretty-printer that loads, parses, and outputs formatted JSON from stdin or files using QJsonDocument, common in API response processors.
  5. Configuration Management
    Develop a settings loader that reads INI-style configs with QSettings and applies them to app behavior (e.g., log level), as seen in configurable CLI deployers.

Advanced Exercises

  1. Networking: HTTP Client
    Construct a simple HTTP GET client using QNetworkAccessManager to fetch and display web content in the terminal, emulating download tools like wget wrappers.
  2. Threading for Concurrent Operations
    Create a multi-threaded file scanner using QThread that processes directories in parallel and reports results via signals, for performance-critical tasks like backup verifiers.
  3. External Process Execution
    Build a task runner that spawns and monitors subprocesses with QProcess (e.g., running shell commands and capturing output), similar to build scripts in CI/CD pipelines.
  4. TCP Server for Local Services
    Implement a basic echo server using QTcpServer that listens on a port and responds to client connections in the console, modeling lightweight local daemons.
  5. Logging and Debugging
    Enhance a data processor with structured logging using QLoggingCategory, outputting levels (debug/info/error) to file or console, essential for production CLI utilities.
  6. Internationalization Support
    Localize a multi-language CLI tool using QTranslator and QString for translatable strings, handling locale switches via args, for global deployment scripts.
  7. Single-Instance Enforcement
    Extend a file locker app to ensure only one instance runs using QSharedMemory or QtSingleCoreApplication, preventing conflicts in shared resource managers.
